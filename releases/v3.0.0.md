# LOOKOUT Release Notes v3.0.0

Release date: 2025-10-13

## Changes

- New Aerial view gives you another 3D perspective for situational awareness. Button available on the top left of the screen for split screen view. Swipe to orbit the camera around the boat
- Stability option uses heading sensor IMU data to detect pitch and roll of the boat for a better augmented reality experience. Toggle on in Boat Set-up preferences
- Many more performance and optimization improvements
